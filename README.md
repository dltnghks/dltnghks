## About Me

안녕하세요. 게임 플레이 경험을 실제 구현으로 연결하는 개발자 이수환입니다.

Unity와 Unreal 기반의 게임 개발을 중심으로 전투 시스템, UI 프레임워크, 데이터 파이프라인, AI 로직을 설계하고 구현해왔습니다. 
기능 단위의 개발에 머무르지 않고, 협업 과정에서 반복적으로 활용할 수 있는 구조와 워크플로우를 만드는 것을 중요하게 생각합니다.

프로젝트에서는 게임 시스템 아키텍처 설계, ScriptableObject 및 CSV 기반 데이터 관리, 행동 트리 기반 AI, 에디터 확장을 활용한 제작 환경 개선 등을 경험했습니다. 또한 플레이어 행동 데이터를 분석하고, 게임 환경과 난이도를 동적으로 조정하는 연구를 수행하며 게임 개발과 AI 연구를 함께 확장해왔습니다.

문제를 구조적으로 이해하고, 구현과 실험을 통해 검증 가능한 결과로 만드는 개발자가 되는 것을 목표로 하고 있습니다.
